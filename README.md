# TrafficLight
Test task for "ITELMA"

## Описание

Консольное приложение, моделирующее систему управления вольерами с животными (Cow, Sheep, Goat).

Каждый вольер имеет состояние светофора:
- 🟢 Green — активный вольер
- 🔴 Red — закрыт
- 🟡 Yellow — переходное состояние

Каждый вольер оснащен заслонкой, которая имеет два состояния:
1. `O` — открыта;
2. `C` — закрыта.

Система работает в непрерывном цикле и поддерживает ручное управление через клавиатуру.

---

## Управление

- `c` — выбрать вольер Cow (Корова)
- `s` — выбрать вольер Sheep (Овцы)
- `g` — выбрать вольер Goat (Козы)
- `w` — включить / выключить режим "Волки"

---

## Логика работы

### Основной режим
- В любой момент активен только один вольер
- Каждые 15 секунд система автоматически обновляет состояние активного вольера
- Переключение между вольерами происходит через переходное состояние (Yellow)

---

### Переключение вольеров
При смене активного вольера:
1. Текущий вольер переходит в Yellow
2. Через 5 секунд происходит переключение
3. Новый вольер становится Green

---

## Режим "Волки"

При активации (`w`):

- Все вольеры закрываются (Red)
- Любые команды управления игнорируются
- Система находится в заблокированном состоянии

При повторном нажатии `w`:

- Блокировка снимается
- Система возвращается к последнему активному вольеру
- Переход выполняется через Yellow-состояние
